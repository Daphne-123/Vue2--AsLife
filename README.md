# Vue2--AsLife

以vue.js和webpack技术实现的移动端软件，写法不复杂，作为学完vue的练手项目。

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
