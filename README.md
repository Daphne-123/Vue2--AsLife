# Vue2--AsLife
&emsp;用vue.js和webpack技术做的移动端软件<br>
以vue2语法为技术实现的一个移动端软件，写法不复杂，作为学完vue的练手项目。

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
